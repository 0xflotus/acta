# Security Policy

## Supported Versions

| Version | Supported          |
| ------- | ------------------ |
| > 3.0   | :white_check_mark: |
| < 3.0   | :x:                |

## Reporting a Vulnerability

To report a vulnerability, send an e-mail to fabien.huet@gmail.com. Avoid creating a public issue as long as it is not resolved.
